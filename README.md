# ✈️ US Flight Delay Analysis — Power BI Dashboard

## 📊 Project Overview

An interactive Power BI dashboard analyzing flight delays across 5 major US airlines in Q1 2019. This project demonstrates advanced data visualization and storytelling using Power BI, complementing a parallel Excel dashboard built with the same dataset to showcase the strengths of each tool.

## 🔍 Key Insights

- Delta Air Lines has the lowest delay rate (14.2%) — best performer
- Southwest has the highest delay rate (21.1%) — despite most flights
- Thursday is the worst day for delays (21.01%)
- Tuesday is the best day to fly (16.32%)
- February is the worst month for delays (21.70%)
- Delays climb steadily through the day — fly early morning for best results
- March has the most flights (368.34K) — spring break surge

## 🛠️ Power BI Features Used

- Multi-row Card — Airline KPI strip across the top
- Gauge Visual — On-Time Performance
- Donut Chart — Market share breakdown
- Conditional Formatting — Gradient colors (green to red) on bar charts
- Slicers — Interactive month filter
- DAX Measures — Custom calculated metrics
- Cross-filtering — Click any visual to filter the entire dashboard

## 📊 Excel vs Power BI Comparison

| Feature | Excel | Power BI |
|---------|-------|----------|
| Interactivity | Static charts | Click-to-filter, slicers |
| KPI Cards | Manual formatting | Dynamic multi-row cards |
| Conditional Colors | Manual per-bar | Automatic gradient rules |
| Filtering | Pivot filters | Cross-filtering all visuals |
| Sharing | File share | Publish to web |
| Mobile | Not responsive | Auto mobile layout |
